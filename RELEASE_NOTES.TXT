11.12.2014 
Added functionality for tracking suite and method owners.
1. Suite owner should be specified in each suite xml file add new parameter on suite level:
<suite verbose="1" name="LCUI Smoke Tests Suite"
	skipfailedinvocationcounts="false" junit="false" parallel="tests" data-provider-thread-count="50" annotations="JDK">
	<parameter name="suiteOwner" value="pkebets"/>


2. Use special jana annotation to specify each TestNG method owner:
	@Test (description="JIRA#AUTO-175")
	@MethodOwner(owner="pkebets")
	@Parameters({"firstName","lastName","email"})
	public void testCheckRatesRates(String firstName, String lastName, String email)

3. Use Ownership static method to retieve information about suite and method owners when necessary:
	String suiteOwner = Ownership.getSuiteOwner(context);
	String mthodOwner = Ownership.getMethodOwner(result);

11.04.2014
Improved core capabilities to work with dynamic elements using @FindBy annotation.

1. Declare extended web element using @FindBy notation and java %s, %d etc 
@FindBy(xpath = "//*[text()='%s']")
private ExtendedWebElement myElement;
2. Use DriverHelper->format method to generate ExtendedWebElement with particular string value:
ExtendedWebElement myDynElement = format(myElement, "my_particular_text");
or
click(format(myElement, "my_particular_text"));

Note: it is possible to provide several patterns
@FindBy(xpath = "//*[text()='%s' and @class='%d']")
private ExtendedWebElement myElement;
ExtendedWebElement myDynElement = format(myElement, "my_particular_text", 15);

10.21.2014
Added file auto-upload feature for Firefox browser. If you configure settings correctly then click onto file automatically saves it into user tmp location.
Two settings should be added into _config.properties to enable the feature:
#boolean value
auto_download=true
#mime type of the files you would like to download automatically
auto_download_apps=application/pdf
